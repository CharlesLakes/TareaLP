Para ejecutar el codigo en linux se debe tener instalado racket con el siguiente comando:
	apt-get install racket
Y se ejecuta con:
	racket nombre_archivo.rkt
Para el caso de la entrega sera:
	racket problemaN.rkt
Tal que n es el numero de el problema.

Para ejecutar el codigo en windows se debe tener instalar racket desde -> https://racket-lang.org/download/
Con esto instaldo se puede ejecutar el codigo de igual forma que en linux pero se tiene que tener agregado racket a la PATH
La otra forma es usar la interfaz grafica de DrRacket y arbrir los archivos de cada problema y ejecutar.
